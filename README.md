# DungeonQuest

## Project Status
This project is actively under development. Features, structure, and gameplay systems will continue to evolve as work progresses.

## Inspiration & Credits
This project is a digital recreation of the dungeon-crawler mechanics found in the HeroQuest board game (originally by Milton Bradley / Avalon Hill).
It draws inspiration from the board layout, turn structure, hero roles, and monster interactions, while introducing original names, UI, and implementation details.
